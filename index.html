Voice Commands

Voice commands allow you to control the cheat sheet via your voice. Once you hit enable, the cheat sheet will begin listening to your voice commands and will persist between resets until you disable it. It will by default read from your default microphone, so ensure that it is set correctly. The voice recognition will continue to work even if the page is not in focus and you are using the same microphone in-game.
Evidence:
  • "difficulty <difficulty>" -> Set the difficulty
  • "number of evidence <num>" -> Custom: Set number of evidence
  • "hunt duration <duration>" -> Custom: Set hunt duration
  • "evidence <evidence>" -> Mark as a found evidence
  • "evidence not <evidence>" -> Mark as not an evidence
  • "evidence clear <evidence>" -> Reset state of an evidence
  • "monkey paw <evidence>" -> This will mark the evidence off with monkey paw logic
Speed:
  • "speed <speed>" -> Mark specified speed
  • "speed clear <speed>" -> Reset state of specified speed
  • "speed [has] line of sight" -> Mark having standard line-of-sight speed up
  • "speed not line of sight" -> Mark no standard line-of-sight speed up
  • "speed clear line of sight" -> Reset line-of-sight filter
Hunt Sanity:
  • "sanity <sanity>" -> Mark specified hunt sanity
  • "sanity clear <sanity>" -> Reset state of specified hunt sanity
Ghosts:
  • "ghost not <ghost>" -> Grey out the specified ghost card
  • "ghost clear <ghost>" -> Reset state of a ghost card (excluding hidden cards)
  • "ghost select <ghost>" -> Select/highlight specified ghost card
  • "ghost remove <ghost>" -> Remove the specified ghost card from the screen
  • "ghost guess <ghost>" -> Mark the ghost as an unconfirmed guess
  • "ghost killed by <ghost>" -> Select the ghost AND mark that it killed you
  • "ghost show|info <ghost>" -> Scroll the specified ghost card into view
  • "ghost tests <ghost>" -> Show the 0 evidence tests for the specified ghost in the guide
Timer:
  • "timer start" -> Start the smudge timer
  • "timer stop " -> Stop the smudge timer
  • "cooldown start" -> Start the cooldown timer
  • "cooldown stop " -> Stop the cooldown timer
  • "hunt start" -> Start the hunt timer
  • "hunt stop " -> Stop the hunt timer
Misc:
  • "show filters" or "show tools" -> Flips the menu to show the filters/tools
  • "show descriptions" or "show behaviors" -> Shows ghost behavior on the ghost cards
  • "hide descriptions" or "hide behaviors" -> Hides ghost behavior on the ghost cards
  • "ghost speed <percent>" -> Set ghost speed modifier to 50%, 75%, 100%, 125%, or 150%
  • "select map <map name>" -> Selects the specified map
  • "show map [<map name>]" -> Opens the map tab and displays map
  • "close map" or "hide map" -> Closes the map tab
  • "reset journal" or "reset cheat sheet" -> Reset the cheat sheet (will save if linked with Discord)
  • "domo move" -> Toggles the side of the screen that Domo is on
  • "stop listening" -> Stop the voice recognition

Data Privacy

Voice commands are routed through the built-in Web Speech API. Although microphone access is required for this feature, your voice is never recorded or stored.

Disclaimers

The ZN-Desktop-Link does not interact with, read from, write to, or modify any Phasmophobia game files, nor does it interact with the game itself. The software is an extension of the Unofficial Phasmophobia Cheat Sheet. As such, the cheat sheet must remain open for the Desktop Link to work. Any issues, bugs, or concerns can be taken care of on our Discord.

Installing

Once the .msi file has downloaded, simply run the installer and it will walk you through the installation process.
During installation, you may see the following popup window:
This popup appears anytime you try to run software that has not been officialy verified by Microsoft (which costs hundreds of dollars per year). The ZN-Desktop-Link is safe to run and does not contain any malicious code. You can bypass this warning and continue the install by clicking "More info" then "Run Anyway".

Linking the Desktop Link

Once the software is installed, there are 2 ways to link the ZN-Desktop-Link:

Manual Link

Navigate to the "Desktop Link" section of the "Settings" tab and click the "Create" button.
Copy the code it generates (in the format of 'DL-#####') and paste that into the "Link ID" field of the "Connect to Cheat Sheet" page in the ZN-Desktop-Link software. Then hit "Connect".

Auto Link

Alternatively, you may navigate to the "Desktop Link" section of the "Settings" tab and click the "Create & Launch" button. This will automatically open the ZN-Desktop-Link and connect it for you.

Usage

HotKeys

The ZN-Desktop-Link provides a variety of hotkeys to control the cheat sheet while in game, allowing you to start timers, mark evidence, see ghost data, and tap out footsteps all without having to "tab out" of the game. The provided hotkeys can be seen and changed on the "Connected!" page in the Desktop Link.

Settings

The ZN-Desktop-Link provides many customizable options for you to make the software the perfect companion to your gameplay. To open the options, click the settings icon on the lower right corner of the "Connected!" page in the Desktop Link. From here you can show/hide the overlay and ghost overlay, change the location, color, and screen of the overlays, and more.

Overlay

Once enabled, the overlay will be displayed on the screen and will be pinned to the front so that it is visible while you are in game. It provides a quick view of the timers, the bpm/speed, the evidences, and the ghost selected/guessed.

Ghost Overlay

In addition to the overlay, there is a second overlay that can be enabled that will show you all the ghosts. From here you can click on a ghost (or cycle through them with hotkeys) to see the behaviors and tells of that ghost.
